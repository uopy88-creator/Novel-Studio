# Novel Studio — Ideas

아직 로드맵에 없거나, 로드맵에 있어도 **구체화 전**인 아이디어를 모아 둔다.  
구현 전에 [PRODUCT.md](./PRODUCT.md) 목표와 맞는지 확인한다.

---

## 집필 보조

| 아이디어 | 설명 |
|----------|------|
| **Word Vault** | 단어 · 관용구 · 말투 샘플 보관. Dialogue Vault와 비슷하지만 “대사”가 아닌 “어휘” |
| **Inspirations** | 이미지·문장·링크·메모형 영감 보드 |
| **Foreshadowing** | 복선 심기/회수, Document와 연결 |
| **Conflict Checker** | 설정·시점·이름 충돌 가벼운 점검 |
| **Timeline** | 사건 순서 시각화 (집필용, 세계관 위키 과다 지양) |

---

## 언어 · 외부 API

| 아이디어 | 설명 |
|----------|------|
| **국립국어원 API** | 표준국어대사전 등 검색 · 뜻풀이 |
| **Dictionary API** | 일반 사전/유의어 (v2.0 Dictionary와 연결) |
| **맞춤법 검사** | 외부 검사기 연동 (선택, 방해되지 않게) |

---

## 계정 · 데이터

| 아이디어 | 설명 |
|----------|------|
| **클라우드 동기화** | Supabase 등. `auth` / `services` / `database` / Storage Layer 구조는 준비됨 |
| **Login** | 이메일 등. 로그인 화면은 아직 없음 |
| **Export** | txt / md 내보내기 |
| **Import** | 기존 원고 파일 가져오기 |
| **백업/복원** | LocalStorage JSON 덤프 |
