# Novel Studio Roadmap

이 문서는 **언제 무엇을 만들지**를 정리한 로드맵이다.  
우선순위가 바뀌면 이 파일을 먼저 수정한 뒤 코드를 맞춘다.

---

## 버전 한눈에 보기

| 버전 | 목표 | 상태 |
|------|------|------|
| v0.1 | 로컬 작업실의 핵심 뼈대 | 진행 중 |
| v0.2 | 집필을 돕는 보조 도구 | 예정 |

## v0.1 — 로컬 작업실 핵심

목표: 브라우저만으로 **작품을 만들고, 문서를 정리하고, 원고를 쓰는** 최소 작업실.

| 기능 | 설명 |
|------|------|
| **Project** | 작품(작업실) 생성·수정·삭제·목록 |
| **Document** | 작품 안의 문서/장 단위 목차 관리 (현재 코드에서는 `Chapter`로 구현) |
| **Manuscript** | Document에 연결된 실제 원고 본문 작성 |
| **Characters** | 인물 카드 관리 |
| **Memo** | 빠른 메모 |

### v0.1에서 하지 않는 것

- 로그인 / 클라우드
- 드래그 앤 드롭 정렬 (위·아래 버튼으로 충분)
- 고급 세계관 위키

---

## v0.2 — 집필 보조 도구

목표: 쓰는 도중에 **꺼내 쓰기 좋은** 도구를 붙인다.

| 기능 | 설명 |
|------|------|
| **Dialogue Vault** | 대사·말투 샘플 보관 |
| **Foreshadowing** | 복선 심기/회수 추적 |
| **Writing Statistics** | 집필량·시간 통계 |

---

## v0.3 — 계정과 이동성

목표: 한 기기에 묶이지 않게 한다.

| 기능 | 설명 |
|------|------|
| **Login** | 사용자 계정 |
| **Cloud Sync** | 작품 데이터 동기화 |
| **Export (txt)** | 원고를 txt로 내보내기 |

---

## v1.0 — Stable Release

목표: 일상 집필에 쓸 수 있을 만큼 **안정적**인 버전.

- v0.1~v0.3 핵심 기능이 동작한다
- 데이터 손실을 막는 기본 안전장치
- 문서(docs)와 실제 코드가 크게 어긋나지 않는다

---

## 진행 원칙

1. **한 번에 하나의 기능** — 여러 기능을 동시에 크게 만들지 않는다.
2. **문서 먼저** — 큰 방향이 바뀌면 `docs/`를 고친 뒤 코드를 맞춘다.
3. **로컬 먼저** — v0.1~v0.2는 LocalStorage 중심으로 검증한다.
4. **세계관보다 집필** — 설정 백과보다 쓰는 흐름을 우선한다.
