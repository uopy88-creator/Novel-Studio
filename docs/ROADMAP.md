# Novel Studio — Roadmap

버전별 목표. 우선순위가 바뀌면 이 파일을 먼저 고친다.

---

## 한눈에 보기

| 버전 | 목표 |
|------|------|
| **v1.0** | 로컬 작업실 핵심 |
| **v1.5** | 집필 보조 도구 확장 |
| **v2.0** | 계정 · 클라우드 · 검색 · 사전 |

---

## v1.0 — 로컬 작업실 핵심

| 기능 | 설명 |
|------|------|
| **Project** | 작품 생성 · 수정 · 삭제 · 목록 |
| **Dashboard** | 분량 · 최근 Document · 메모/캐릭터 수 (보기 전용) |
| **Documents** | 문서(목차) 관리, 종류(소설/시/에세이/기타) |
| **Editor** | Manuscript 원고 작성 · 자동 저장 · 검색 · 실시간 통계 |
| **Characters** | 인물 카드 |
| **Dialogue Vault** | 대사 보관 · 태그 · 즐겨찾기 |
| **Memo** | 빠른 메모 |

---

## v1.5 — 집필 보조 확장

| 기능 | 설명 |
|------|------|
| **Word Vault** | 단어 · 표현 · 어휘 보관 |
| **Inspirations** | 영감 · 스크랩 · 레퍼런스 |
| **Foreshadowing** | 복선 심기 / 회수 추적 |

---

## v2.0 — 계정과 확장

| 기능 | 설명 |
|------|------|
| **Login** | 사용자 인증 |
| **Cloud Sync** | 작품 데이터 동기화 (Supabase 등) |
| **Search** | 작품 전체 · 크로스 Document 검색 |
| **Dictionary API** | 사전/국어원 등 외부 API 연동 |

---

## 진행 원칙

1. 한 번에 하나의 큰 기능
2. 기존 기능을 깨지 않는 추가·수정
3. 로컬(LocalStorage)을 먼저 검증한 뒤 클라우드로 확장
4. UI는 미니멀 유지 ([DESIGN.md](./DESIGN.md))
